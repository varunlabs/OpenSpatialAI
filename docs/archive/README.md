# Documentation Archive

This folder contains older drafts, reference LaTeX sources, and superseded outputs.

## Folders

- `drafts/`: previous Markdown drafts and missing-input notes.
- `latex_reference/`: earlier LaTeX paper source and generated reference outputs.

## Usage Rule

Files in this folder are not the primary current submission files. Use them only for reference, comparison, or recovery.

Current files should live in:

```text
docs/final_submission/
docs/project_manual/
```
