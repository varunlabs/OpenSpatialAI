# Project Overview

## Introduction

This project implements a behavior-centered multimodal framework for real-time context modeling in Extended Reality (XR).

## Objective

To detect user context in real time using lightweight signals available in XR systems without relying on specialized hardware.

## Key Idea

User context is inferred as a behavioral state based on:

* Head orientation
* Motion patterns
* Interaction signals

## Context States

The system detects four states:

* Engaged
* Exploring
* Distracted
* Idle

## Scope

This implementation represents a working prototype validated in a Unity simulation environment.
